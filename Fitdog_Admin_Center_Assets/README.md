# Fitdog Admin Center Assets

Upload the `public` folder into the root of your Next.js app in Cursor.

## Included
- 43 admin/dashboard SVG icons
- React TypeScript icon components
- Admin Center hero image
- Admin action card graphics
- Admin activity graph graphic
- System status graphic
- Admin badges
- Sidebar/background graphics
- Fitdog admin logo
- Admin avatar placeholder

## Recommended upload path

Copy this folder into your project:

```txt
public/assets/admin/
```

Then use assets like:

```jsx
<img src="/assets/admin/icons/svg/admin-center.svg" />
<img src="/assets/admin/illustrations/admin-center-hero.svg" />
<img src="/assets/admin/logos/fitdog-admin-logo.svg" />
```

## Cursor instruction

After uploading, tell Cursor:
Use `/assets/admin/...` paths for the Admin Center page and connect these icons/illustrations to the new admin dashboard cards, sidebar, profile settings, system health, maintenance, audit logs, password reset, and user role management screens.
