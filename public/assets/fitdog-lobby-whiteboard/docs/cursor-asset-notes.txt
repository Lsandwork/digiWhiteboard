Cursor implementation note:

Place this folder at:
public/assets/fitdog-lobby-whiteboard/

Use these assets when building:
/lobby/checkouts?display=tv
/admin/lobby

Do not hard-code these placeholder dog images as real data. Use them only as fallback assets when Gingr profile photos are missing.

Preferred fallback mapping:
dog_photo_url || "/assets/fitdog-lobby-whiteboard/dogs/png/dog-profile-beau-golden.png"

Use SVG icons whenever possible for crisp 4K display.
Use PNG fallback only where SVG is inconvenient.

Critical safety:
The lobby route must never call Gingr directly from the frontend. It should call only internal routes like:
GET /api/lobby/checkouts
GET /api/lobby/promotions
GET /api/lobby/events
GET /api/lobby/status

Polling interval must be 10 seconds minimum, preferably 15 seconds.
